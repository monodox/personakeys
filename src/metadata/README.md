# PersonaKeys Icon

This directory should contain the plugin icon for marketplace distribution.

## Icon Requirements

**File**: `icon.png` (or other supported format)  
**Size**: 256x256 pixels (recommended)  
**Format**: PNG with transparency support  
**Usage**: Displayed in Logitech Marketplace and Loupedeck Marketplace

## Current Status

⚠️ **Placeholder icon needed**

The project currently uses placeholder icons. For marketplace submission, create a custom 256x256px icon representing PersonaKeys.

## Design Guidelines

- **Theme**: Hardware + AI fusion
- **Elements**: Consider incorporating:
  - Physical button/ring imagery
  - AI/neural network motifs
  - Developer-focused symbols (code brackets, etc.)
- **Colors**: Match the PersonaKeys brand (purple gradient: #667eea → #764ba2)
- **Style**: Modern, clean, recognizable at small sizes

## Creating the Icon

1. Design a 256x256px icon
2. Save as `icon.png` in this directory
3. Ensure it's included in the build output
4. Test visibility in Logitech Options+ / Loupedeck software
